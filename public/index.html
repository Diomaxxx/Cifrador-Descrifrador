Cifrador Diogo

Esta aplicación te permite cifrar y descifrar textos utilizando una tabla ASCII personalizada. Puedes cargar un archivo de texto, seleccionar la longitud del cifrado, y guardar el texto cifrado junto con la tabla ASCII generada. También puedes cargar archivos previamente cifrados para descifrarlos utilizando la tabla ASCII correspondiente.
